# Triumph Assistant Boundary

## What Phase 16 does

Triumph Assistant answers how-to questions by searching a bundled, reviewed product-help
catalog. It covers getting started, recording, device and latency setup, audio editing, MIDI,
Keys, VST3 instruments and freeze, mixer routing, tempo and automation, AI Producer, export,
project safety, shortcuts, availability, and basic troubleshooting.

Help or F1 opens the panel. Quick-topic buttons enter common questions, and a ranked Related
Help selector exposes alternate matches. When an article corresponds to an existing control,
the user may explicitly open that real control from the answer.

## What Phase 16 does not do

- It does not send the question, project, audio, MIDI, device, or identity data anywhere.
- It does not use an online chat, search, account, analytics, or generative-model service.
- It does not claim to understand unrestricted questions outside the bundled help catalog.
- It does not invent a feature or show fake cloud, collaboration, publishing, or marketplace UI.
- It does not execute a project edit merely because a question matched an article.

The article catalog is source-controlled product documentation. New answers must be reviewed
against implemented behavior and covered by deterministic search tests before release.
